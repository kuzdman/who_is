2 - Обосноавние выбора темы
	В наше время машинное обучение помогает в решении многих задач, в том числе с распознаванием объектов на изображениях.
	Используемые технологии позволят пользователям быстрее находить фотографии в накопленных архивах.

3 - Назначение программы
	Программа позволяет распознавать лица людей в архиве фотографий с использованием алгоритмов машинного обучения.

4 - Используемые библиотеки и компоненты
	4.1 dlib - библиотека для обработки лиц на основе машинного обучения (http://dlib.net)
	4.2 PyQt5 - набор «привязок» графического фреймворка Qt для языка программирования Python, выполненный в виде расширения Python (https://pypi.org/project/PyQt5)
	4.3 scikit-image - это библиотека Python, предназначенная для обработки изображений (https://scikit-image.org)
	4.4 scipy - библиотека для языка программирования Python с открытым исходным кодом, предназначенная для выполнения научных и инженерных расчётов (https://scipy.org)
	4.6 flask - фреймворк для создания веб-приложений на языке программирования Python (http://flask.pocoo.org)
	4.5 Anaconda - open-source дистрибутив языка Python, R, C/C++ и других, который позволяет обрабатывать данные, производить вычисления, а также упрощает работу с пакетами (https://www.anaconda.com)
	4.6 Bootstrap - инструментарий для создания сайтов и веб приложений (https://getbootstrap.com)
	4.7 dlib_face_recognition_resnet_model_v1.dat - модель для выделения лиц по 68 ключевым точкам (https://github.com/ageitgey/face_recognition_models/blob/master/face_recognition_models/models/dlib_face_recognition_resnet_model_v1.dat)
	4.8 shape_predictor_68_face_landmarks.dat - нейронная сеть ResNet34 для распознавания лиц (https://github.com/ageitgey/face_recognition_models/blob/master/face_recognition_models/models/shape_predictor_68_face_landmarks.dat)

6 - Правила использования программы
	Сначала необходимо добавить изображения с лицами людей, которые нужно распознать в архиве фотографий. Для этого нужно выбрать фотографию с лицом человека и указать его имя (кнопка "Add Person").
	Далее можно распознавать архивы фотографий. Для этого нужно указать каталог фотографий и имя распознаваемого человека (кнопка "Recognition").
	По окончанию процесса распознавания появляется ссылка на веб ресурс, в котором отображаются результаты распознавания.
